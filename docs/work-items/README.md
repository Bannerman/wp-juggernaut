## Adding a Work Item

Create a new file: `WI-{NNN}-{short-slug}.md` with this template:

```markdown
# WI-{NNN}: Title

## Status: To Do

## Priority: High | Medium | Low

## Summary

Brief description of the task.

## Tasks

- [ ] Task 1
- [ ] Task 2
```

Then add a row to the table above.

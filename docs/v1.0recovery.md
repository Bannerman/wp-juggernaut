                 │ 12→## Core Concepts                                                                                                                                             
                 │ 13→                                                                                                                                                             
                 │ 14→### 1. The Core App                                                                                                                                          
                 │ 15→The base application that handles:                                                                                                                           
                 │ 16→- WordPress REST API communication                                                                                                                           
                 │ 17→- Authentication (Application Passwords)                                                                                                                     
                 │ 18→- Standard WordPress content (posts, pages, custom post types)                                                                                               
                 │ 19→- Standard taxonomies (categories, tags, custom taxonomies)                                                                                                  
                 │ 20→- Basic fields (title, content, excerpt, featured image, status, slug)                                                                                       
                 │ 21→- Local SQLite caching and dirty-state tracking                                                                                                              
                 │ 22→- Push/Pull synchronization                                                                                                                                  
                 │ 23→                                                                                                                                                             
                 │ 24→**The Core knows nothing about MetaBox, SEOPress, ACF, or any other WordPress plugin.**                                                                      
                 │ 25→                                                                                                                                                             
                 │ 26→### 2. Plugins                                                                                                                                               
                 │ 27→Plugins extend the Core to add support for **WordPress plugins**. Each plugin:                                                                               
                 │ 28→- Registers new field types, tabs, or UI components                                                                                                          
                 │ 29→- Defines how to read/write data for that WordPress plugin's API                                                                                             
                 │ 30→- Can be enabled/disabled per site profile                                                                                                                   
                 │ 31→                                                                                                                                                             
                 │ 32→**Example Plugins:**                                                                                                                                         
                 │ 33→                                                                                                                                                             
                 │ 34→| Plugin | Adds Support For | Features |                                                                                                                     
                 │ 35→|--------|------------------|----------|                                                                                                                     
                 │ 36→| `metabox` | MetaBox (MB) | Custom meta fields, field groups, repeaters |                                                                                   
                 │ 37→| `seopress` | SEOPress | SEO title, meta description, social, robots |                                                                                      
                 │ 38→| `acf` | Advanced Custom Fields | ACF field groups and field types |                                                                                        
                 │ 39→| `woocommerce` | WooCommerce | Product management, variations, pricing |                                                                                    
                 │ 40→| `yoast` | Yoast SEO | Alternative SEO plugin support |                                                                                                     
                 │ 41→                                                                                                                                                             
                 │ 42→### 3. Site Profiles                                                                                                                                         
                 │ 43→Configuration files that define:                                                                                                                             
                 │ 44→- Site connection details (URL, name)                                                                                                                        
                 │ 45→- Which post types to manage                                                                                                                                 
                 │ 46→- Which plugins to enable                                                                                                                                    
                 │ 47→- Field mappings and customizations                                                                                                                          
                 │ 48→- UI preferences                                                                                                                                             
                 │ 49→

## 2026-03-24 - Adding ARIA labels to icon-only buttons\n**Learning:** Icon-only buttons without an `aria-label` are not properly read by screen readers. For accessible UX, every icon button such as '<X />', '<Trash2 />', etc., must be explicitly labeled using `aria-label` so users can understand their purpose.\n**Action:** Always include a descriptive `aria-label` attribute on `<button>` elements where the only child content is an icon (e.g. from lucide-react).
